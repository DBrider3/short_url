# Django ShortURL Clone Coding

## Getting Start
### django project 시작하기
1. `python -m venv .venv`
    1. Python 3.11.6 version
    2. Pip 24.0 version
2. `source .venv/bin/activate`
3. `pip install --upgrade pip`
4. `pip install -r requirements.txt`
